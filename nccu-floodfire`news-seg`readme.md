## NewsSeg

 * 將新聞標題與內容斷詞，並計算每日詞頻
 * 展示：http://news-seg.source.today/

## How to start

 * Download a daily redis db from [here](http://news.source.today/redis.rdb) and import to your redis server (md5sum: aa9491e99e5f31706ea5f8e6f81e5f2f)
 * `make init`
 * `php artisan serve`
